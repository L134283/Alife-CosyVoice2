# Alife-CosyVoice2
# 🎤 CosyVoice2 语音合成插件（Alife）

基于 CosyVoice2 API 的高质量语音合成插件，用于 Alife 桌宠 / 角色语音输出。

---

# 🚀 功能特点

- 高质量 TTS 语音合成
- 支持自定义音色（Speaker ID）
- 支持语速调节
- 支持本地 CosyVoice2 API
- 自动缓存语音文件（避免重复生成）

---

# 📦 安装方法

## 1️⃣ 安装插件

在 Alife 插件市场中搜索：
CosyVoice2语音合成


点击安装并启用。

---

## 2️⃣ 启动 CosyVoice2 服务

确保你已经运行 CosyVoice2 API 服务：
http://127.0.0.1:9981


如果服务未启动，插件无法生成语音。

---

# ⚙️ 配置说明（重要）

在 Alife 配置界面中，你会看到如下配置项：

{
  "ApiUrl": "http://127.0.0.1:9981",
  "ReferenceAudio": "67",
  "Speed": 1.0
}

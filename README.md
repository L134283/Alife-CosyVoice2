# 📘 CosyVoice2 语音合成插件 v1.0.1

基于 **CosyVoice2 API** 的语音合成插件，支持 UI 配置、自定义服务路径、自动服务管理与稳定性增强。

---

# ✨ 功能特性

* 🎤 高质量语音合成（CosyVoice2）
* 🧩 可视化 UI 配置面板
* 📁 自定义服务路径（Python / 模型 / 项目目录）
* 🚀 自动启动后端服务
* 🛡️ 健康检查 + 异常自动重启
* 🔁 请求重试机制
* ⚡ 本地缓存优化响应速度

# ⚙️ 安装方式

## 1️⃣ 获取插件

方式一（推荐）：

* 在 Alife 插件市场直接下载

方式二（手动下载）：

* 前往 Release 下载：

👉 `CosyVoice2 v1.0.1 - Alife.Function.Speech.CosyVoiceTTS.zip`

---

## 2️⃣ 安装插件

解压到插件目录：

```text
Plugins/
└── Alife.Function.Speech.CosyVoiceTTS/
```

---

## 3️⃣ 配置服务（可选）

在 UI 或配置文件中设置：

* Python 路径
* 模型路径
* 服务启动目录
* API路径端口
* 其他项音色、语速等
---

# 🚀 快速开始

1. 启动 Alife 桌宠系统
2. 打开语音插件配置界面
3. 设置 CosyVoice2 服务路径
4. 点击启动桌宠
5. 自动启动服务，可能需要等待一分钟
6. 开始正常使用

---

# 🧠 架构说明

```text
Alife 桌宠
   ↓
插件层
   ↓
CosyVoice2 API 服务
   ↓
语音输出
```

---

# 🔧 配置说明

| 配置项          | 说明          |
| ------------ | ----------- |
| python_path  | Python 执行路径 |
| model_path   | 模型目录        |
| service_path | 服务启动目录      |
| auto_restart | 是否自动重启      |
| retry_count  | 请求失败重试次数    |

---

# ⚠️ 注意事项

* 首次使用需确保 CosyVoice2 服务正常启动
* 建议使用 GPU 环境以提升生成速度
* Windows 下建议使用管理员权限运行

---

# 📌 依赖

* Alife 桌宠框架
* CosyVoice2 API 服务
* Python 3.8+

---

# 💡 常见问题

## ❓ 无法启动服务？

请检查：

* Python 路径是否正确
* 端口是否被占用

---

## ❓ 没有声音输出？

请检查：

* 模型是否加载成功
* API 服务是否运行
